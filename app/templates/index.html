Upload an Image for Classification

This is a simple image classifier using a pre-trained model trained on the CIFAR-10 dataset.
You can upload an image and see the predicted class.
It can recognize:

- Airplane
- Automobile
- Bird
- Cat
- Deer
- Dog
- Frog
- Horse
- Ship
- Truck

Note that this is a simple model and may not always be accurate (It's trash). -Splatchy
